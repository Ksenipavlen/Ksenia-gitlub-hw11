Задание 2
2.1 Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?
SYN (-sS): Отправляет пакет с флагом SYN (запрос на соединение). Это стандартный, но «полуоткрытый» метод.
FIN (-sF) и Xmas (-sX): Отправляют пакеты с «неправильными» флагами (FIN, PSH, URG), которые не используются для установки соединения.
UDP (-sU): Отправляет пустой UDP-пакет. Трафик не имеет флагов, как в TCP.
2.2 Как отвечает сервер?
На SYN-сканирование:
Если порт открыт, сервер отвечает пакетом SYN/ACK.
Если порт закрыт, сервер отвечает пакетом RST.
На FIN и Xmas-сканирование:
Если порт открыт, сервер игнорирует пакет и не отвечает ничего.
Если порт закрыт, сервер отвечает пакетом RST.
На UDP-сканирование:
Если порт закрыт, сервер обычно отвечает сообщением ICMP Port Unreachable.
Если порт открыт или фильтруется брандмауэром, сервер, как правило, не отвечает ничего.
